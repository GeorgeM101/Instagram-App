### INSTAGRAM-CLONE

Developer: George Mboya

#### PROJECT DECRIPTION

Website where users can upload pictures, follow friends, like and comment on posts.

#### USER STORIES

A user should be able to

- Create an account
- Login an existing account
- Post photos
- Comment and like on posts

#### TECHNOLOGIES USED

- Python 3.8
- Django Version 4.0.3
